# struts2和springmvc区别？

Struts2是一个请求一个Action,SpringMVC是单例模式的

Struts2是基于类的横切，SpringMVC基于方法（可以和上条结合起来思考）粒度更细

一个入口是Filter，一个入口是Servlet。则两者个方面机制不同，比如拦截器实现机制

SpringMVC开发速度和性能较优于Struts2，流程更容易理解

Spring体系更加强大和活跃

Struts2更符合面向对象的实现
